While iframes are a pretty common way to allow for third-party widgets and embedded content, it's rare you'll have to write your own iframe tags. Even in this example you'd be better off going to the YouTube video's page and copying and pasting the embed code into your HTML file. This makes sure you are using the most current version of YouTube's code rather than something custom that might lose compatibility over time.
